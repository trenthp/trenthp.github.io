The Brief

This project was made to be submitted to Eastern Washington University as an option to chaning their main student portal page. It is meant to give a quick one-stop-shop for all needed links, school news, student snapshots, etc. I built this using rapid prototyping techniques.

The Approach

I started with putting together a persona for a typical student at this school. I moved on to build a style tile based off of questions from students who fit this profile. Then came the sketching mappings for information hierarchy.

The Result

The school would still be able to disperse important information to the student, but this first needed to leverage the key information for which a student visits a portal page. Financial information, student records, and key links to class information is all readily available on the main landing page.
